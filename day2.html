Python Syntax, Loops, Conditions

Basic Syntax

Python is known for its clean, readable syntax that uses indentation to define code blocks instead of braces or keywords.

Hello World Example
# This is a comment
print("Hello, World!")  # This prints a message

# Variables don't need type declarations
name = "Python"
version = 3.10
is_awesome = True

print(f"{name} {version} is awesome: {is_awesome}")

Key points about Python syntax:

  • Indentation (usually 4 spaces) defines code blocks
  • Comments start with #
  • Variables are dynamically typed
  • Statements don't need semicolons
  • String formatting uses f-strings (in modern Python)

Conditional Statements

Conditional statements let your program make decisions based on conditions.

If-Elif-Else Example
# Basic if-else structure
temperature = 25

if temperature > 30:
    print("It's hot outside!")
elif temperature > 20:
    print("It's a nice day!")
else:
    print("It's a bit cold.")

# Logical operators: and, or, not
is_sunny = True
is_weekend = False

if is_sunny and not is_weekend:
    print("It's a sunny weekday!")
elif is_sunny or is_weekend:
    print("At least one good thing!")
else:
    print("Neither sunny nor weekend.")

Python's conditional statements include:

  • if: Executes code if a condition is true
  • elif: Checks another condition if previous conditions were false
  • else: Executes code if all conditions are false
  • Logical operators: and, or, not
  • Comparison operators: ==, !=, >, <, >=, <=

Loops

Loops allow you to repeat code execution multiple times.

For Loops
While Loops
For Loop Examples
# Looping through a range
for i in range(5):  # 0 to 4
    print(i, end=" ")  # Prints: 0 1 2 3 4

print()  # New line

# Looping through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}s")

# Enumerate gives index and value
for index, fruit in enumerate(fruits):
    print(f"Fruit {index+1}: {fruit}")

# Loop with conditional
for num in range(1, 11):
    if num % 2 == 0:
        print(f"{num} is even")
    else:
        print(f"{num} is odd")

For loops in Python can iterate over:

  • Ranges of numbers using range()
  • Lists, tuples, strings, and other iterables
  • Dictionary keys, values, or items

The enumerate() function is useful when you need both the index and value.

While Loop Examples
# Basic while loop
count = 0
while count < 5:
    print(count, end=" ")
    count += 1  # Don't forget to increment!
# Prints: 0 1 2 3 4

print()  # New line

# While loop with break
number = 0
while True:  # Infinite loop
    number += 1
    if number > 10:
        break  # Exit the loop
    if number % 2 == 0:
        continue  # Skip even numbers
    print(number, end=" ")  # Prints: 1 3 5 7 9

print()  # New line

# While with else (runs when condition becomes false)
attempts = 0
while attempts < 3:
    print(f"Attempt {attempts+1}")
    attempts += 1
else:
    print("All attempts completed")

While loops continue as long as a condition is true. Important concepts:

  • break: Exits the loop immediately
  • continue: Skips the rest of the current iteration
  • else clause: Executes when the loop condition becomes false (not after a break)
  • Be careful to avoid infinite loops by ensuring the condition will eventually become false

Functions, Lists, Dictionaries

Functions

Functions are reusable blocks of code that perform specific tasks. They help organize code and avoid repetition.

Function Examples
# Basic function
def greet(name):
    """This function greets the person passed in as a parameter"""
    return f"Hello, {name}!"

# Call the function
message = greet("Alice")
print(message)  # Prints: Hello, Alice!

# Function with default parameters
def power(base, exponent=2):
    """Returns base raised to the power of exponent"""
    return base ** exponent

print(power(3))      # Prints: 9 (3^2)
print(power(2, 3))   # Prints: 8 (2^3)

# Function with multiple return values
def get_min_max(numbers):
    """Returns the minimum and maximum values from a list"""
    return min(numbers), max(numbers)

min_val, max_val = get_min_max([5, 3, 8, 1, 9])
print(f"Min: {min_val}, Max: {max_val}")  # Prints: Min: 1, Max: 9

# Lambda function (anonymous function)
square = lambda x: x * x
print(square(4))  # Prints: 16

Key concepts for Python functions:

  • Defined with the def keyword
  • Can have docstrings (documentation strings) enclosed in triple quotes
  • Can have default parameter values
  • Can return multiple values (as a tuple)
  • Lambda functions are small anonymous functions defined with the lambda keyword
  • Functions are first-class objects (can be passed as arguments, returned from other functions)

Lists

Lists are ordered, mutable collections that can store items of different types.

List Operations
# Creating lists
empty_list = []
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# Accessing elements (indexing starts at 0)
print(numbers[0])    # Prints: 1
print(numbers[-1])   # Prints: 5 (negative indices count from the end)

# Slicing
print(numbers[1:3])  # Prints: [2, 3] (start at index 1, end before index 3)
print(numbers[:3])   # Prints: [1, 2, 3] (start at beginning, end before index 3)
print(numbers[2:])   # Prints: [3, 4, 5] (start at index 2, go to the end)

# Modifying lists
numbers.append(6)    # Add to the end: [1, 2, 3, 4, 5, 6]
numbers.insert(0, 0) # Insert at index 0: [0, 1, 2, 3, 4, 5, 6]
numbers.remove(3)    # Remove first occurrence of 3: [0, 1, 2, 4, 5, 6]
popped = numbers.pop()  # Remove and return last item: 6, list becomes [0, 1, 2, 4, 5]
numbers[0] = 10      # Modify element: [10, 1, 2, 4, 5]

# List operations
a = [1, 2, 3]
b = [4, 5]
c = a + b           # Concatenation: [1, 2, 3, 4, 5]
print(len(a))       # Length: 3
print(3 in a)       # Membership test: True
print(a * 2)        # Repetition: [1, 2, 3, 1, 2, 3]

# List comprehensions
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]
even_squares = [x**2 for x in range(10) if x % 2 == 0]  # [0, 4, 16, 36, 64]

Important list methods and operations:

  • append(): Add an item to the end
  • insert(): Add an item at a specific position
  • remove(): Remove the first occurrence of a value
  • pop(): Remove and return an item (default: last item)
  • sort(): Sort the list in-place
  • reverse(): Reverse the list in-place
  • List comprehensions provide a concise way to create lists

Dictionaries

Dictionaries are unordered collections of key-value pairs, providing fast lookups by key.

Dictionary Operations
# Creating dictionaries
empty_dict = {}
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Accessing values
print(person["name"])  # Prints: John

# Using get() (safer, returns None or default if key doesn't exist)
print(person.get("age"))  # Prints: 30
print(person.get("email", "Not found"))  # Prints: Not found

# Modifying dictionaries
person["email"] = "john@example.com"  # Add new key-value pair
person["age"] = 31  # Modify existing value
del person["city"]  # Remove a key-value pair

# Dictionary methods
keys = person.keys()    # Get all keys
values = person.values()  # Get all values
items = person.items()  # Get all key-value pairs as tuples

# Checking if a key exists
if "name" in person:
    print("Name exists")

# Dictionary comprehension
squares = {x: x**2 for x in range(5)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Nested dictionaries
contacts = {
    "John": {"phone": "555-1234", "email": "john@example.com"},
    "Mary": {"phone": "555-5678", "email": "mary@example.com"}
}

print(contacts["John"]["phone"])  # Prints: 555-1234

Key dictionary concepts:

  • Keys must be immutable (strings, numbers, tuples, but not lists)
  • Values can be any type
  • get() method provides safe access with a default return value
  • keys(), values(), and items() return view objects
  • Dictionary comprehensions provide a concise way to create dictionaries
  • Dictionaries can be nested for complex data structures

Package Management (pip, virtualenv)

pip - Python Package Installer

pip is the standard package manager for Python, used to install and manage packages from the Python Package Index (PyPI).

Common pip Commands
# Install a package
pip install package_name

# Install a specific version
pip install package_name==1.2.3

# Upgrade a package
pip install --upgrade package_name

# Uninstall a package
pip uninstall package_name

# List installed packages
pip list

# Show information about a package
pip show package_name

# Save dependencies to a file
pip freeze > requirements.txt

# Install dependencies from a file
pip install -r requirements.txt

Important pip concepts:

  • pip comes pre-installed with Python 3.4+ and Python 2.7.9+
  • You can specify version constraints: == (exact), >= (minimum), <= (maximum)
  • requirements.txt files are commonly used to share and reproduce environments
  • pip installs packages globally by default (which is why virtual environments are important)

virtualenv - Virtual Environments

Virtual environments create isolated Python environments for projects, allowing different projects to have different dependencies.

Working with Virtual Environments
# Install virtualenv (if not already installed)
pip install virtualenv

# Create a virtual environment
virtualenv myenv

# On Windows, activate the virtual environment
myenv\Scripts\activate

# On macOS/Linux, activate the virtual environment
source myenv/bin/activate

# Your terminal prompt should change to show the active environment
# (myenv) $

# Install packages (they'll be isolated to this environment)
pip install numpy pandas

# Deactivate the virtual environment when done
deactivate

# Using Python's built-in venv module (Python 3.3+)
python -m venv myenv

# The activation commands are the same as with virtualenv

Benefits of virtual environments:

  • Isolate dependencies for different projects
  • Avoid version conflicts between packages
  • Test applications with different Python versions
  • Keep your global Python installation clean
  • Make projects reproducible across different machines

Modern Alternatives

While virtualenv is still widely used, there are modern alternatives:

  • venv: Built into Python 3.3+ (use with python -m venv)
  • conda: Package and environment manager, popular in data science
  • pipenv: Combines pip and virtualenv functionality
  • poetry: Modern dependency management and packaging

Daily Task

Write a Python script to take user input and print a response.

For today's task, you'll create a simple interactive Python script that:

  1. Asks the user for their name
  2. Asks the user for their age
  3. Asks the user for their favorite programming language
  4. Prints a personalized response based on their inputs
Example Solution
# Daily Task: User Input and Response

def get_user_input():
    """Get input from the user and return as a dictionary"""
    name = input("What is your name? ")
    
    # Use a loop to ensure we get a valid age
    while True:
        try:
            age = int(input("How old are you? "))
            break  # Exit the loop if conversion to int succeeds
        except ValueError:
            print("Please enter a valid number for your age.")
    
    language = input("What's your favorite programming language? ")
    
    return {
        "name": name,
        "age": age,
        "language": language
    }

def generate_response(user_data):
    """Generate a personalized response based on user data"""
    name = user_data["name"]
    age = user_data["age"]
    language = user_data["language"]
    
    # Create different responses based on the data
    response = f"Nice to meet you, {name}! "
    
    if age < 18:
        response += "You're quite young to be programming already! "
    elif age < 30:
        response += "You're at a great age for learning programming. "
    else:
        response += "It's never too late to code! "
    
    if language.lower() == "python":
        response += "I see you have excellent taste in programming languages! Python is my favorite too."
    else:
        response += f"{language} is a great language! Have you considered trying Python as well?"
    
    return response

def main():
    print("Welcome to the Python Interaction Program!")
    print("----------------------------------------")
    
    # Get user input
    user_data = get_user_input()
    
    # Generate and display response
    response = generate_response(user_data)
    print("\nResponse:")
    print(response)
    
    # Additional information based on age
    years_coding = user_data["age"] - 10
    if years_coding > 0:
        print(f"\nIf you started coding at age 10, you could have {years_coding} years of experience by now!")

if __name__ == "__main__":
    main()

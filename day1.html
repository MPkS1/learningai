Day 1: Introduction to AI & Machine Learning

What is Artificial Intelligence?

Artificial Intelligence (AI) refers to computer systems designed to perform tasks that typically require human intelligence. These include learning from experience, recognizing patterns, understanding language, making decisions, and solving problems.

Real-World Examples:

  • Voice assistants like Siri, Alexa, and Google Assistant
  • Recommendation systems on Netflix and Spotify
  • Self-driving cars
  • Chess-playing computers like Deep Blue
  • Content generation tools like ChatGPT and DALL-E

How Does AI Work?

At its core, modern AI works by:

  • Processing large amounts of data
  • Identifying patterns in that data
  • Making predictions or decisions based on those patterns
  • Improving over time through more data and feedback

Narrow AI vs. General AI

Narrow AI (Weak AI)

Systems designed to perform a specific task very well, but cannot perform outside their programmed domain.

Examples:

  • Chess-playing programs
  • Image recognition software
  • Spam filters
  • Virtual assistants

Current Status: Widely deployed and used daily

General AI (Strong AI)

Systems with human-like intelligence that can understand, learn, and apply knowledge across different domains.

Examples:

  • Science fiction concepts like HAL 9000 from "2001: A Space Odyssey"
  • Theoretical systems that can reason across domains like humans

Current Status: Does not yet exist; remains theoretical

Machine Learning vs. Deep Learning

Machine Learning and Deep Learning are subfields of AI, with Deep Learning being a specialized type of Machine Learning.

Machine Learning

A subset of AI where algorithms learn from data without being explicitly programmed for specific tasks.

Key Characteristics:

  • Requires feature engineering (humans deciding what features are important)
  • Works well with structured data
  • Typically requires less data and computing power

Common Algorithms:

  • Linear Regression
  • Decision Trees
  • Random Forest
  • Support Vector Machines
  • K-means Clustering

Deep Learning

A subset of Machine Learning using neural networks with many layers (hence "deep").

Key Characteristics:

  • Automatic feature extraction (the system learns what's important)
  • Excels with unstructured data (images, text, audio)
  • Requires large amounts of data and computing power

Common Architectures:

  • Convolutional Neural Networks (CNN)
  • Recurrent Neural Networks (RNN)
  • Transformers
  • Generative Adversarial Networks (GAN)

Real-World Example: Image Recognition

Traditional Machine Learning Approach: Engineers would define features like edges, corners, and color histograms, then train algorithms to recognize patterns in these manually defined features.

Deep Learning Approach: A neural network learns hierarchical features automatically - first identifying edges, then shapes, then higher-level features like "eyes" or "wheels" - without human-defined features.

Language Models (GPT, BERT, etc.)

Language models are AI systems trained to understand and generate human language. They have revolutionized natural language processing tasks in recent years.

How Language Models Work

Modern language models use neural networks (particularly transformer architectures) trained on vast amounts of text to:

  • Predict the next word in a sequence
  • Understand relationships between words
  • Capture contextual meaning
  • Generate coherent and contextually appropriate text

Major Language Models

GPT (Generative Pre-trained Transformer)

Developed by OpenAI, the GPT family specializes in generating human-like text based on prompts.

Key Features:

  • Autoregressive (predicts next token based on previous tokens)
  • Excels at text generation and completion tasks
  • Used in tools like ChatGPT
  • Has evolved through versions (GPT-1, GPT-2, GPT-3, GPT-4)

Example Application: Writing assistance, content creation, conversational AI

BERT (Bidirectional Encoder Representations from Transformers)

Developed by Google, BERT introduced bidirectional training of transformers.

Key Features:

  • Bidirectional (considers context from both before and after each word)
  • Excels at understanding meaning and context
  • Primarily used for understanding rather than generation

Example Application: Google Search, sentiment analysis, document classification

Other Notable Language Models

  • T5 (Text-to-Text Transfer Transformer): Treats all NLP tasks as text-to-text problems
  • LLaMA: Meta's open-source large language model
  • Claude: Anthropic's language model, designed with a focus on helpfulness and harmlessness
  • PaLM: Google's Pathways Language Model

Impact of Language Models

Language models have transformed how we interact with computers, enabling:

  • More natural human-computer interaction
  • Automated content creation
  • Advanced translation services
  • Code generation and assistance
  • Automated summarization and information extraction
